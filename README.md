# AI-Agent

## Screenshot omni-parser

This repo includes a small “omni parser” that accepts a screenshot and outputs:

- **`parsed.json`**: OCR text + bounding boxes (and heuristic UI boxes)

### Setup

```bash
python3 -m venv .venv
. .venv/bin/activate
pip install -r requirements.txt
```


### Run

```bash
. .venv/bin/activate
python main.py
```
